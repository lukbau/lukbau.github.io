Introduction

OpenNetlistView is a tool for viewing netlists in the Yosys JSON format. It is designed to run as an native application and a WASM application in a web browser.

The project was developed as part of a master thesis. The finished thesis can be found here.

Note: The current version is a prototype. The generation of schematics for bigger and more complex designs can be very slow.

Information on how to install and use OpenNetlistView can be found in the following sections.

A good starting point is the Prebuilt / AppImage section.
